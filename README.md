# Home Upkeep

Reminders for the house stuff you always forget, and every appliance's details in one place. Pick from about 60 common maintenance tasks with sensible schedules, mark one done with a tap, and the next due date sets itself. Keep each appliance's brand, model, and serial number handy, and jump straight to a search for its manual.

No paywall. No ads. No tracking. No accounts.

## What it does

- A starter library of common maintenance tasks (HVAC filter, gutters, water heater, smoke detectors) with editable intervals
- One-tap done with undo; the schedule rolls itself from the day you actually did the work
- One local reminder when a task is due. No streaks, no nagging
- An appliance registry: brand, model, serial, purchase year, notes
- Find manual: one tap opens a search for your exact model's documentation
- Export and import all your data as a file, any time
- Dark mode, large-text support, and translations

## Run

```bash
npm install
npm run ios      # or: npm run android
```

## Privacy

Your data stays with you. No accounts, no tracking, no analytics. Reminders are scheduled locally on the device. See [PRIVACY.md](PRIVACY.md).
